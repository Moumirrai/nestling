## Gotická architektura

- 12 - 14. století
- vzink ve Francii
- období vrchoného a pozdního středověku
- Název „gotika“ zavedli italští humanisté, kteří ji mylně spojovali s Góty, jejichž umění bylo považováno za barbarské
